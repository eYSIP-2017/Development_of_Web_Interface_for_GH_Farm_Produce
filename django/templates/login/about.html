About the Project:

We are designing a web portal to be used for monitoring and keeping a record of vegetable produce in the greenhouse. Our project logs crop data in greenhouse to reduce manual labour. Initially, the farm produce (i.e vegetables) are weighed on a weighing machine. The weighing machine communicates with an server and sends pre-defined characteristics of the vegetables like quantity(according to the weight), its image etc which is then stored in the database and updated on the web portal. The portal also aims at implementing an e-Commerce store. It involves python, for backend of the weighing machine.

Website will be designed using the following Web development technologies:

  • Front-end+Back-end: AngularJS, python, Bootstrap, Javascript (including JQuery and AJAX)
  • Framework: Django
  • Database: MySQL
  • Web Tools: Sublime Text, Pycharm Community Edition
  • Development platform: Linux Mint
  • Target Environment: Desktop, Tablets, Mobiles

System has three components:

  1. Weighing machine.
  2. Ethernet connection with server.
  3. Web interface for data logging and monitoring
